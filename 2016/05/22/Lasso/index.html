The Art of Lasso

May 22, 2016 | Lasso | Hits

Even though I got to know lasso at about seven months ago and have implemented this algorithm in the machine learning courses, I cannot say I know lasso. I cannot explain why this algorithm can actually select the important variables instead of being sparse. Also, do you you know why lasso is called lasso? So this blog is used for organizing the knowledge of lasso. I will introduce the formulas of lasso(including general lasso, group lasso, fused lasso, graphical lasso), some ways to solve lasso and the reason why it makes sense.
Oh, BY THE WAY, lasso is short for least absolute shrinkage and selection operator.

Background:
Lasso is a variable selection method with the $l_1$ penalty. Its assumption is that only a few variables metter. It’s important in high-dimensional cases(n<p). Also it can be used to do signal recovery. Compressive sensing is based on lasso(another interesting signal recovery method is called matrix factorization, based on the assumption that matrix is low-rank). The formula of lasso is:
$$min ||y- X\beta ||_2^2 + \lambda ||\beta||_1$$

Besides the traditional lasso, there are general lasso, group lasso, fused lasso, graphical lasso etc. Here is a good package SLEP.

Knowledge from machine learning

In the machine learning course, what I learnt are the basic lasso, the way to solve lasso(including lar and ADMM), the general lasso and graphical lasso.

What’s the difference between lasso and ridge regression
Ridge and lasso are all penalties for avoiding overfitting. Ridge regression penalizes large weights, while lasso penalizes non-zero weights.

The basic lasso will be more discussed in the paper below(Fan and Li 2001) so is not covered here.

Graph-guided fused LASSO:
$$min ||y- X\beta ||_2^2 + \lambda ||\beta||_1 + \lambda_2\underset{s,t}{\Sigma}|\beta_s-\beta_j|, \quad if (s,t) is an edge in graph$$
This is used in this case: you have a prior knowledge that some variables are neighbours, so their coefficent are more likely to be similar.

General LASSO:
$$min ||y- X\beta ||_2^2 + \lambda ||D\beta||_1 $$
This can be very flexible. You can change the matrix $D$ to generate fused lasso, linear trend filtering, quadratic trend filtering(just make every variables to be connected with the variables two-step besides them).

LASSO SOLVER: LARS
LARS is based on LAR and is used for computing regularization path of LASSO. LAR is an efficient stepwise variable selection algorithm. It’s based on the greedy idea: First, pick a most correlated variable, then add a new variable step-wisely. So the ‘active’ subset $\mathcal{A}$ increases all the time.(This can cause some problems)

LASSO SOLVER: Shooting
Shooting algorithm is another lasso solver and is based on stochastic coordinate descent.

LASSO SOLVER: ADMM(Alternating Directions Method of Multipliers)
ADMM is a very cool method based on my opinion. It’s not hard to understand but it shows a very powerful idea of finding optimal solution. An example is that you want to minimize $f(x)+g(x)$, you can re-write it to $f(x)+g(z),\quad s.t. \quad x=z$. Then write the augmented Lagrangian:$L_p(x,y,z) = f(x)+g(z)+y^T(x-z)+\frac{p}{2}||x-z||^2$. Here the $l_2$ term destorys the separability of Lagrangian!!!
The update rule for ADMM is that $x = argmin(L), z = argmin(L), y = y + p(x-z)$. Update until convergence.

Variable Selection via Nonconcave Penalized likelihood and its Oracle properties

motivation:
Traditional variable selection methods such as stepwise deletion and subset selection ignore stochastic errors and theoretical properties are hard to understand. Also are time-consuming.

contribution:

  1. Demonstrate the rates of convergence
  2. Prove that it select the right model. That is, when the true parameters have some zero components, they are estimated as 0 with probability tending to 1.Inspiring, this is what I need!!!

methodology

$$\frac{1}{2} ||y- X\beta||_2^2 + \lambda \underset{j=1}{\overset{d}{\Sigma}} p_j(|\beta_j|) = \frac{1}{2} ||y - \hat y||^2 + \frac{1}{2} \underset{j=1}{\overset{d}{\Sigma}}(z_j-\beta_j)^2 + \lambda \underset{j=1}{\overset{d}{\Sigma}} p_j(|\beta_j|)$$

Here $Z = X^Ty, \hat y = XX^Ty, p_j(\cdot)$ is a general penalty function.
Here note that we normalize the X!!!!!, that means $X^TX=I$. That’s why if you want to use lasso, you need to normalize it ahead
