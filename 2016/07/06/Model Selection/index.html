Model Selection

Model selection methods are used when you have a set of candidate models or your models have some undicided parameters. In this case, you need to use the model selection mothod. Most of the contents are from wiki. But more contents will be added after reading some to-do list.

Model selection is a task of selecting a statistical model from a set of candidate models given data. Usually, a good model balances the goodness of fit and the model complexity(means avoiding overfitting). I get contacted with AIC, BIC, MDL and Cross Validation at this time point.

Akaike information criterion(AIC)

$$AIC = -2ln(L) + 2k$$
L – Likeihood function
k – the number of estimated parameters
Does not provide null hypothesis, so can tell nothing about the quality of the model in a absolute sense

AICc: AIC with a correction for finite sample sizes
$$AICc = AIC + \frac{2(k+1)k}{n-k-1}$$
Assumption: model is univariate, linear, has normally-distributed residuals

Bayesian information criterion(BIC)

$$BIC = -2ln(\hat L) + kln(n)$$
$\hat L$ – the maximized value of likelihood function of model
n – sample size
k – number of free parameters

Limitation:

1.n has to be much larger thant k
2. Cannot handle high dimension(a little related to 1)

AIC and BIC

  1. AIC has therotical advantages over BIC
  2. Under the assumption of the exact “true model” is not in candidate set, AIC is asymptotically optimal in selecting model with least mean squared error, BIC is not.
  3. BIC has larger penalty on model complexity than AIC

MInimum description length/Algorithmic information theory(MDL)

$$MDL = \hat{NLL} + \frac{\hat \theta}{2}log(n)$$
$\hat{NLL}$ – minimized negtive likelihood(with $\hat \theta$)
