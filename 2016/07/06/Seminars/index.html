It’s a pity that I lost some useful seminars’ notes. From next quarter I will think of updating the notes here in case of lost… UW-seattle is a great place for research communication. I really enjoy these seminars.

Venkat chandrasekaran: Latent-variable graphical modeling via convex optimization

math, caltech
Motivation is with the hidden variables, the estimate of the graphical model will be denser than the actual network. By we can prove that it equals the sparse true network + a low rank matrix. This can be formed as a convex problem.

Michel gallery: Learning to Converse: An End-to-End Neural Approach

Microsoft research, natural language process
Not too much about the models, mainly use user - based LSTM model. But the motivation is interesting.
Normally a robot responds to a question without connecting to the conversation. Also, it doesn’t make sense to respond like “where to eat? Restaurant!”. The answer should be based on the location, weather etc. Also, personal assistant is needed.

Sham Kakade: Big Data Regime

UW CSE
A very clear and explicit explanation of GD, SGD and some other algorithms. The most valuable information I get is that the SGD is unbiased so cannot use constant learning rate. But it’s fast and good for big data. So improve it by duality or change the learning process.

Poster Session: Convex Optimization and Big data

I attend these two sessions. Overall it’s interesting. I can talk about the idea with others(finally ==!). There are three interestin topics:

  1. Update rule for network. Usually the update rule is to update the variables step by step without consider the relationship between variables. So one deep learning group learn use the correlation matrix to improve the update accuracy and speed
  2. Optimize un-directed graphical model by convex optimization. I think this is very related to michel gallery’s topic.
  3. Objective track with HMM. We can consider the movements of the objective as the markov chain. And this person does a very interesting thing: Use optimal flow to train the HMM!!! Even though he hasn’t got any results now LOL
