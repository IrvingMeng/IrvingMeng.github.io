Deep Causal Model

Causal model is useful to discover the connections between variables so it can be of great importance for us to understand the data generation mechanism(For example, find the root cause). And deep learning becomes extremely popular nowadays and it can have great performance. The key advantages of deep learning is scalability, ?, ?(I don’t remember… Ask Atlas later). It can deal with more complex problem compared to traditional methods. For example, the health care problem(Normally, the health-care data is irregular, noisy and sparse).

Also, there are some things we can do: Deep learning for time-series data, causal model for time-series(This is very very interesting).

Before you think it’s appealing, ask yourself: Do I really need to use this? A good researcher won’t be 为了用而用(I don’t know how to translate, LOL).
