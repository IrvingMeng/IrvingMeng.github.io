Application

This blog summarizes several applications of Directed Acyclic Graph (DAG) model.

Risk assessment for hypertension and hypertension complications incidences using a Bayesian network

Contributions: This paper uses BN network to model the hypertension and hypertension complications incidences. First Cox regression method as well as domain knowledge are used to select variables. Then the data are discretized and sub-sampled to build the network. In the end several validation methods are implemented including domain knowledge, the probability inference and the accuracy, sensitivity, AUC. The contributions are that it implements the BN to this problem…

Structure learning:

  1. Constraint-based algorithms: have lower time complexity and better scalability
  2. Score-based algorithms: more accurate and stable
  3. This paper use Max-min Hill-Climbing (MMHC) algorithm, which combines constrain-based and score-based algorithms. It first use a conditional independency test to find a set of adjacent nodes for each node (constraint-based) and then performs a standard hill-climbing search to orient the edges between nodes (score-based).

Structure learning: MMHC algorithm, Causal explorer toolkit(Aliferis et al. 2003)
Inference: Bayes Net toolbox(Murphy 2002)

For writing:

  1. Key contributions. This needs to be done before your writing.
  2. Data construction and variable definition:
    • Give a table of the data name and detail
    • Explain how to deal with pre-processing and missing value
  3. Variable Selection:
    • Purely by data
    • By domain knowledge(explain why, the reason is obvious. A network may have many layers. So one can not only consider the relationship with the last layer)
  4. Discretization:
    • Explain why you need to do this(BN methods cannot be applied easily to mixed data)
    • For better interpretation, this paper only use 2-3 levels!!!!!!!!!
    • A table of subjects report each characteristic’s name, number of each level and percent of the level.
  5. Unbalance: Can be eliminate this problem, using under-sampling. This need to see whether it’s needed.

  6. Validation:

    • Accuracy, sensitivity and specificity
    • AUC (area under ROC)
    • In this paper, the authors’ result shows that BN cannot beat Cox regression. They explain the reason is that BN uses the discrete data. And BN tends to be more sensitive.I agree with this interpretation.But whey don’t use the same dataset then compared with other methods?
  7. Inference
    • The authors offer several tables to show that the network make sense. For example, use the three nodes and write the P(A|B,C) when B, C takes different values, P(A|B,C) shows significant differences.

Combining data and meta-analysis to build Bayesian networks for clinical decision support

Ideas for writing:

  1. In general, a multivariate model can be built in a number of ways: (1) purely from data using statistical and machine learning techniques, (2) from a combination of clinical knowledge and data or (3) from published literature using multivariate meta-analysis
    techniques, (4) this paper proposes a method of combining the results of meta-analyses, clinical knowledge and data to pro-vide decision support for complex decision problems where the data is scarce.
  2. Challenges: Complex, important but rare(hard to collect very large datasets).

Useful sentences:

  1. Complex clinical decisions require the decision maker to evaluate multiple factors that may interact with each other. Many clinical studies, however, report univariate relations between a single factor and outcome.
  2. The mechanistic relations between the observed factors and outcome may depend on clinical factors that are not available in the data or not examined in the meta-analysis. For example, the data may not distinguish between the measurements and true state of a variable, or may exclude a part of the important causal factors in the domain. In this case, latent variables are used to model clinical knowledge in the BN.
