Directed Acyclic Graph

Posted on 2016-07-12   |   In Theory

Directed Acyclic Graph(DAG) is especially useful to model causality. But it’s hard to find the global optimal solution since the time complexity is exponential and most algorithms are using greedy method(this maybe not true, it’s just my idea now). This blog summarizes some papers to help organize the idea.

To Read

  1. 2006, Meinshausen and Buhlmann. High dimensional graphs and variable selection with lasso. Why l1 can recovers the undirected network structure
  2. 2006, Wainwright. Inferring graphical model structure using l1-regularized pseudo-likelihood.extend the formal prove to logistic regression

Learning Graphical Model Structure using L1-Regularization Paths

Based on this paper, there is a matlab package: DAGlearn

motivation

  1. Learning structure from data is an NP-hard problem
  2. If you want you model to be suitable for density estimation and prediction, estimate the parameters(not solved in this paper)
  3. Each node should have different penalty since some nodes have high degree(connectivity)
  4. Undirected model cannot show causality

contribution

  1. propose a new heuristic technique to improve the performance of existing DAG-learn method(max-min hill-climbing MMHC, max-min parents-children MMPC, order search OS). Use lasso to search for potential parents first, L1MB(L1-regularized Markov Blanket).
  2. use different $\lambda$ for different nodes.

methodology
Lasso + MDL. See figure 1

Validation
Number of erroneously removed data, structural errors, MDL of the networks.
